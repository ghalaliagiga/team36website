Entry 5

During the development phase of our project website, we recognized the importance of familiarising ourselves with the SketchUp and AutoCAD software. This was necessary as we needed to be able to effectively navigate the workspace and perform various functions. To ensure that we had a solid understanding of these programs, we took the time to make a list of all the commands used for each feature. This allowed us to implement the functionality for each feature separately and efficiently.

In addition to familiarising ourselves with SketchUp and AutoCAD, we also spent time getting familiar with the MotionInput source code. This was a completely new area for us, and it was crucial that we understood how it worked in order to make progress on our project. By taking the time to understand these programs, this ensured we were able to begin the designing and implementation process.
