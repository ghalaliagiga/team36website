User Manual

MotionInput can be run from the file MFC-UCL-MI3-Settings.exe. From here they can decide how they want their session to be:

  1. They can choose their CAD application. This will change what speech commands are available and the default dictionary at launch to be optimised for them.
    1. A dictionary is the list of allowed words a user can say.
    2. To change dictionary a user needs to say change dictionary default or change dictionary programname.
  2. They can choose their Tracking Mode. This will let them control their cursor with either their hands or nose.
    1. They can edit their dominant hand if hand mode is selected.
    2. They can change between touch points are regular mouse clicks is selected, however touchpoints is recommended.
    3. They can change the pinch gesture to fit their desired sensitivity.
    4. If nose mode is selected they can choose if they want to use speech or gestures for mouse clicks.
    5. If gestures is picked they can change the functionality of specific gestures to match their liking, e.g. change smile to right click.
    6. They can also change the speed and distance at which the mouse cursor moves.
  3. The user can also edit speech phrases. This will open up a separate dialog. The use just needs to click on the key of the command that they want to edit, and change it to a new word. This will be saved when the user presses okay, and the new user added words will be added to the dictionary.

From here the user can press okay and launch MotionInput. The software will be tailored to their specific CAD software.
